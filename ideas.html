MoltenSheets
A lightweight but powerful spreadsheet web app. Focus on blazing fast data entry, basic formulas, and beautiful offline charting.
  • Dynamic grid logic (A1 + B2)
  • Instant SVG Charting embedded
  • Filter, Sort, and highlighting
  • CSV / Excel offline exporting
MoltenTasks (or MoltenBoard)
A Kanban-style project management board like Trello, allowing dragging cards across columns natively.
  • Drag & Drop columns and cards
  • Checklists and due dates inside cards
  • Kanban, List, and Calendar views
  • Custom color tags and priority levels
MoltenMind
A visual mind-mapping and brainstorming tool, using a node-based infinite canvas structure.
  • Infinite panning/zooming HTML5 Canvas
  • Connecting nodes with Bezier curves
  • Rich text nodes (images, links, bold)
  • Clean auto-layout algorithms
MoltenMail (Template Builder)
A beautiful email template designer that outputs raw, perfectly responsive HTML you can paste into Gmail/Outlook.
  • Drag and drop layout blocks
  • 100% inline-CSS generation for mail clients
  • Variables and placeholders
  • Preview on mobile, tablet, and desktop
MoltenCode
A secure, local-only code snippet manager. Save, tag, and quickly copy code blocks with beautiful syntax highlighting.
  • Syntax highlighting for 50+ languages
  • 1-click clipboard copy
  • Multi-file snippets and variations
  • Fully searchable by tag, lang, or content
